HTML Tanıtım

Nested (İç İçe): HTML elementleri birbirinin içine yerleştirilebilir. Örneğin <body> içinde <p>, onun içinde <strong> olabilir.

Empty (Boş) Elementler: Kapanış etiketi olmayan kendi kendine yeten elementler:

  • <br> (Break): Metin içinde alt satıra geçmek için kullanılır.
  • <hr> (Horizontal Rule): Konuları ayırmak için yatay çizgi çeker.

Temel Etiketler

1

<!DOCTYPE html> → Tarayıcıya "bu dosya HTML5 standardında yazılmış" der. Sayfanın ilk satırı olmak zorundadır.

2

<html> → Tüm sayfanın en dış kabıdır. Başka her şey bu etiketin içine yazılır.

3

<head> → Sayfa hakkında meta bilgiler tutar: başlık, dil, favicon, CSS bağlantıları gibi. Kullanıcı bu kısmı görmez.

4

<body> → Kullanıcının tarayıcıda gördüğü tüm içerik bu etiketin içine yazılır.

Meta ve Base Etiketleri

<meta>

Meta: Sayfa hakkında görünmeyen ama tarayıcı ve Google için önemli bilgiler veren etiket.

<head>
  <meta charset="UTF-8">                    <!-- Türkçe karakter desteği -->
  <meta name="viewport" content="width=device-width, initial-scale=1.0">  <!-- Mobil uyum -->
  <meta name="description" content="Web tasarım ders notları">  <!-- Google açıklaması -->
  <meta name="author" content="Umut">       <!-- Sayfa yazarı -->
</head>
<base>

Base: Sayfadaki tüm linkler ve dosyalar için varsayılan bir adres belirler.

<head>
  <base href="https://www.example.com/resimler/" target="_blank">
</head>
<body>
  <!-- Bu link aslında https://www.example.com/resimler/logo.png olarak çalışır -->
  <img src="logo.png" alt="Logo">

  <!-- Bu link de https://www.example.com/resimler/hakkimizda.html olarak açılır -->
  <a href="hakkimizda.html">Hakkımızda</a>
</body>

HTML Yorum Satırı

HTML kodları arasına tarayıcı tarafından çalıştırılmayacak notlar eklemek için kullanılır. Kodun okunabilirliğini artırır.

<!-- Bu bir HTML yorum satırıdır. Tarayıcıda görünmez -->
<p>Bu paragraf ekranda listelenir.</p>

HTML Linkler

HTML'de bağlantılar <a> (anchor) etiketi ile oluşturulur. href özniteliği gidilecek adresi belirtir:

<a href="https://umutozmn.github.io/mis-doc/">Bu bir linkdir</a>
  • <a> etiketi: "anchor" kelimesinin kısaltmasıdır — sayfayı bir adrese bağlar.
  • href: "hyper reference" kısaltmasıdır, gidilecek URL'yi içerir.

HTML Images

Resim göstermek için <img> etiketi kullanılır. Kapanış etiketi yoktur. src resmin yolunu, alt ise açıklama metnini belirtir:

<img src="https://via.placeholder.com/150" alt="Örnek resim">

Headings & hr

<hr> Etiketi: Sayfaya yatay çizgi çeker. Konular arasında görsel ayraç oluşturur.

<h1>Birinci Bölüm</h1>
<hr>
<h2>İkinci Bölüm</h2>
Elements & Öznitelikler

Bir HTML etiketinin birden fazla özniteliği (attribute) olabilir. Öznitelik değerleri her zaman tırnak içinde yazılmalıdır: <etiket öznitelik="değer">

Öznitelikler

style Özniteliği: Bir HTML elementine doğrudan CSS stil kuralları uygulamak için kullanılır.

<p style="color:red; font-size:18px;">Bu yazı kırmızı ve büyük.</p>

lang Özniteliği: Sayfanın dilini tarayıcıya ve arama motorlarına bildirir.

<html lang="tr">

title Özniteliği: Mouse ile elementin üzerine gelindiğinde küçük bir tooltip gösterir.

<p title="Bu bir ipucu">
  Farenizi buraya getirin.
</p>

HTML Favicon

Favicon, tarayıcı sekmesinde sitenin adının yanında görünen küçük simgedir. <head> içine eklenir. Boyutu genellikle 16x16 veya 32x32 pikseldir. .ico, .png veya .svg formatlarında olabilir.

<head>
  <link rel="icon" type="image/x-icon" href="resimler/favicon.ico">
</head>

rel="icon" — Bu linkin bir ikon olduğunu söyler.
type="image/x-icon" — Dosya türünü belirtir.
href="..." — Dosyanın yolunu gösterir. Genelde projenin kök klasöründe durur.

HTML Comments

Kodun içine not düşmek için kullanılır. Tarayıcı bu satırları görmezden gelir.

<!-- Bu bir yorum satırıdır -->
<p>Bu yazı görünür.</p>
HTML Styles

Her CSS özelliğinin sonuna noktalı virgül (;) konur. Birden fazla özellik varsa hepsi aynı tırnak içinde yazılır.

Temel Özellikler

background-color — Elementin arkasını boyar.

<h1 style="background-color:powderblue;">This is a heading</h1>
<p style="background-color:tomato;">This is a paragraph.</p>

color — Yazının rengini değiştirir.

<h1 style="color:blue;">Mavi Başlık</h1>
<p style="color:#e74c3c;">HEX ile kırmızı yazı</p>

font-family — Yazının fontunu değiştirir.

<p style="font-family: 'Times New Roman', Georgia, serif;">
  Bu yazı öncelikle Times New Roman, yoksa Georgia.
</p>

font-size — Yazının boyutunu belirler.

<p style="font-size:14px;">Normal boyut (14px)</p>
<p style="font-size:200%;">İki kat büyük (%200)</p>

text-align — Yazının yatay hizasını belirler.

<h1 style="text-align:center;">Ortalanmış Başlık</h1>
<p style="text-align:right;">Sağa yaslanmış paragraf</p>

CSS Bağlama Yöntemleri

External CSS — CSS kuralları ayrı bir .css dosyasına yazılır ve <link> ile bağlanır. En yaygın yöntemdir.

<head>
  <link rel="stylesheet" href="stil.css">
</head>

Internal CSS — <head> içine <style> bloğu ile yazılır. Yalnızca o sayfayı etkiler.

<head>
  <style>
    body { background-color: linen; }
    h1 { color: maroon; }
  </style>
</head>
<body>
  <h1>Internal CSS Test Başlığı</h1>
  <p>Bu alan stil şablonunu test etmek içindir.</p>
</body>

Inline CSS — Doğrudan etiketin içine style="..." ile yazılır. Yalnızca o elementi etkiler.

<h1 style="color:blue; text-align:center;">Mavi ve ortalı.</h1>

Öncelik Sırası: Inline CSS → ID → Class → Element seçici. Aynı seviyede iki kural varsa daha sonra yazılan geçerlidir.

Background Image

background-repeat — Arka plan resmi elementten küçükse tarayıcı onu fayans gibi döşer.

  • no-repeat: Resim yalnızca bir kez gösterilir.
  • repeat-x / repeat-y: Yalnızca yatayda / dikeyde tekrarlar.
<body style="
  background-image: url('https://www.w3schools.com/css/img_5terre.jpg');
  background-repeat: no-repeat;
  background-size: cover;
">
  <p style="color:white; font-size:24px;">Arka plan resmi</p>
</body>

background-attachment — Sayfayı kaydırırken arka planın kayıp kaymayacağını belirler.

  • scroll (Varsayılan): Resim sayfayla birlikte kayar.
  • fixed: Resim ekrana çakılı kalır — paralaks efekti.

Border Stilleri

<p style="border: 2px solid black;">solid — düz çizgi</p>
<p style="border: 2px dotted black;">dotted — noktalı</p>
<p style="border: 2px dashed black;">dashed — kesik kesik</p>
<p style="border: 4px double black;">double — çift çizgi</p>

Margin & Padding

Margin (Dış Boşluk)

Elementin dışında, komşu elementlerle arasındaki boşluktur.

p { margin: 20px; }
Padding (İç Boşluk)

Elementin içinde, içerik ile kenarlık arasındaki boşluktur.

p { padding: 15px; }
<div style="background-color:lightblue; border:5px solid navy; padding:40px; margin:40px;">
  Bu yazı kutunun içinde. 40px padding (içeri), 40px margin (dışarı).
</div>
Text Formatting

Kalın Yazı Etiketleri

<b> Bold

Yazıyı görsel olarak kalın yapar. Anlam taşımaz, salt tasarım amacıyla kullanılır.

<strong>

Yazıyı kalın yapar ve bu metnin önemli olduğunu anlamsal olarak belirtir.

<p>Bu <b>kalın ama önemsiz</b> metin.</p>
<p>Bu <strong>kalın ve önemli</strong> metin.</p>

Eğik (İtalik) Yazı Etiketleri

<i> Italic

Yazıyı görsel olarak eğik yapar.

<em> Emphasis

Yazıyı eğik yapar ve o kelimeye vurgu katar.

<p><i>İtalik</i> görünüm, <em>vurgulu</em> anlam.</p>

Küçük ve İşaretli Metinler

<small>

Yazıyı bir adım daha küçük gösterir. Dipnotlar için idealdir.

<mark>

Metni fosforlu kalem ile işaretlenmiş gibi vurgular.

<p>Normal metin <small>küçük dipnot</small></p>
<p>Sınavda <mark>bu konu çıkacak</mark> dedi hoca.</p>

Silinmiş ve Eklenmiş Metinler

<del> Deleted

Metnin üstüne çizgi çeker. İndirimli fiyat gösteriminde kullanılır.

<ins> Inserted

Metnin altını çizer. Sonradan eklenen bilgileri işaret eder.

<p>Fiyat: <del>250 TL</del> <ins>199 TL</ins></p>

Alt ve Üst Simgeler

<sub> Subscript

Yazıyı satırın biraz altına konumlandırır. (H₂O)

<sup> Superscript

Yazıyı satırın biraz üstüne konumlandırır. (2³)

<p>Su: H<sub>2</sub>O</p>
<p>Alan: 5 m<sup>2</sup></p>
<p>2<sup>10</sup> = 1024</p>
Quotations

<blockquote> — Blok Alıntı: Başka bir kaynaktan alınan uzun bir pasajı belirtmek için kullanılır.

<blockquote cite="https://tr.wikiquote.org">
  Hayatta en hakiki mürşit ilimdir.
</blockquote>

<q> — Kısa Satır İçi Alıntı: Cümle içinde geçen kısa alıntılar için kullanılır. Tarayıcı otomatik tırnak ekler.

<p>Atatürk şöyle demiştir: <q>Hayatta en hakiki mürşit ilimdir.</q></p>

<abbr> — Kısaltma: title özniteliğine tam açıklama yazılır.

<p><abbr title="World Health Organization">WHO</abbr> 1948'de kuruldu.</p>

<bdo> — Metin Yönünü Değiştirme: Metnin akış yönünü tersine çevirir.

<p>Normal: Merhaba Dünya</p>
<p><bdo dir="rtl">Merhaba Dünya</bdo></p>

dir="rtl" → right to left. dir="ltr" → left to right (varsayılan).

Colors

Renk, HTML'de üç farklı yerde uygulanır: background-color, color ve border-color.

Kenarlık Rengi (border):

<h1 style="border: 3px solid Tomato;">Kırmızı kenarlıklı başlık</h1>
<p style="border: 2px dashed DodgerBlue;">Mavi kesik kenarlıklı paragraf</p>

Renk Değerleri

RGB

Kırmızı, Yeşil ve Mavi ışığın karışımı. Her değer 0-255 arasında.

Örnek: rgb(255, 99, 71) → domates kırmızısı

HEX

Web'de en yaygın format. # işaretinin ardından 6 karakterlik kod.

Örnek: #ff6347 → domates kırmızısı

HSL

Renk Tonu (0-360°), Doygunluk (%) ve Parlaklık (%). %0 siyah, %100 beyaz, %50 saf renk.

Örnek: hsl(9, 100%, 64%) → domates kırmızısı

<p style="color: rgb(255,99,71);">RGB ile domates rengi</p>
<p style="color: #ff6347;">HEX ile domates rengi</p>
<p style="color: hsl(9,100%,64%);">HSL ile domates rengi</p>
Links

Target Özelliği

  • _self (Varsayılan): Aynı sekmede açar.
  • _blank: Yeni bir sekme veya pencerede açar.
<a href="https://google.com" target="_blank">Google'ı yeni sekmede aç</a>

URL Türleri

Mutlak URL

Başka bir siteye giden tam adres. Mutlaka https:// ile başlar.

Göreli URL

Kendi sitenizin içindeki başka bir sayfaya giden kısaltılmış adres.

Mailto — E-posta Linki

href değerinin başına mailto: yazılırsa, link tıklandığında kullanıcının varsayılan e-posta uygulaması açılır ve "Kime" alanı otomatik dolu gelir. Kullanıcının bilgisayarında bir mail programı kurulu olması gerekir (Outlook, Apple Mail vs.).

<a href="mailto:umut@example.com">Bana mail at</a>

Konu ve mesaj da önceden doldurulabilir:

<a href="mailto:umut@example.com?subject=Merhaba&body=Selam, sana yazmak istedim.">
  Hazır konulu mail gönder
</a>

mailto: → e-posta adresinin önüne gelir.
?subject= → mail konusunu önceden doldurur.
&body= → mail içeriğini önceden yazar.
Boşluk yerine URL'de %20 kullanılır.

Varsayılan Link Renkleri

  • Ziyaret edilmemiş: Mavi ve altı çizili.
  • Ziyaret edilmiş: Mor ve altı çizili.
  • Aktif (tıklama anında): Kırmızı ve altı çizili.

HTML Bookmarks (Sayfa İçi Bağlantılar)

Uzun sayfalarda belirli bir bölüme doğrudan kaydırma yapmak için kullanılır.

1. Adım: Hedef elemente benzersiz bir id niteliği verin.

2. Adım: Bağlantı etiketinin href özniteliğine #id_adi yazarak linki oluşturun.

<p><a href="#bolum3">Doğrudan Bölüm 3'e Git</a></p>

<h2>Bölüm 1</h2>
<p>İçerik alanı...</p>

<h2>Bölüm 2</h2>
<p>İçerik alanı...</p>

<h2 id="bolum3">Bölüm 3 (Hedef Alan)</h2>
<p>Doğrudan bu bölüme atladınız!</p>
Images

Image Map — Tıklanabilir Resim Haritası

Tek bir resmin farklı bölgelerine farklı linkler atamak için kullanılır.

1

<img usemap="#isim">: Resmi gösterir ve bir haritaya bağlar.

2

<map name="isim">: Haritayı tanımlar.

3

<area>: Tıklanabilir bölgeleri tanımlar.

<img src="https://www.w3schools.com/html/workplace.jpg"
     usemap="#calisma-alani" alt="Çalışma Alanı" width="400" height="379">

<map name="calisma-alani">
  <area shape="rect" coords="34,44,270,350"
        href="https://www.w3schools.com" alt="Bilgisayar" target="_blank">
  <area shape="circle" coords="337,300,44"
        href="https://www.google.com" alt="Kahve" target="_blank">
</map>

Picture Elementi — Duyarlı Resimler

<picture>: Farklı ekran boyutları için farklı resimler göstermek amacıyla kullanılır. Mobil kullanıcılar gereksiz büyük resimleri indirmez.

<picture>
  <source media="(max-width:600px)"
          srcset="https://www.w3schools.com/css/img_5terre_wide.jpg">
  <img src="https://www.w3schools.com/css/img_5terre.jpg"
       alt="Beş Arazi" style="width:100%">
</picture>
Tables

Tablo yapısı: <table> → <tr> (satır) → <td> veya <th> (hücre).

1

<table>: Tablonun tamamını saran ana kapsayıcı.

2

<tr> (Table Row): Her bir satırı tanımlar.

3

<td> (Table Data): Satırın içindeki her bir veri hücresini tanımlar.

4

<th> (Table Header): Başlık hücresidir. Otomatik olarak kalın ve ortalanmış gösterilir.

<table border="1">
  <tr><th>Ad</th><th>Soyad</th><th>Yaş</th></tr>
  <tr><td>Umut</td><td>Yılmaz</td><td>21</td></tr>
  <tr><td>Ayşe</td><td>Kaya</td><td>23</td></tr>
</table>

Thead, Tbody, Tfoot — Tablo Bölümleri

Büyük tablolarda satırları mantıksal gruplara ayırmak için kullanılır. Görsel fark yaratmaz ama kodu okunabilir kılar ve CSS uygulamayı kolaylaştırır.

<thead>

Tablonun başlık satırlarını gruplar. Genellikle <th> hücreleri içerir.

<tbody>

Tablonun asıl veri satırlarını gruplar. Tüm data burada yer alır.

<tfoot>

Tablonun alt satırını gruplar. Toplamlar, özet bilgiler için kullanılır. Tarayıcı bunu her zaman en alta koyar, HTML'de nereye yazsanız yazın.

<style>
  table { border-collapse: collapse; width: 100%; }
  th, td { border: 1px solid #ccc; padding: 10px; text-align: left; }
  thead { background-color: #4a90d9; color: white; }
  tbody tr:nth-child(even) { background-color: #f5f5f5; }
  tfoot { background-color: #e8e8e8; font-weight: bold; }
</style>

<table>
  <thead>
    <tr>
      <th>Ürün</th>
      <th>Adet</th>
      <th>Fiyat</th>
    </tr>
  </thead>

  <tbody>
    <tr><td>Kalem</td><td>3</td><td>15 TL</td></tr>
    <tr><td>Defter</td><td>2</td><td>40 TL</td></tr>
    <tr><td>Silgi</td><td>5</td><td>10 TL</td></tr>
  </tbody>

  <tfoot>
    <tr>
      <td colspan="2">TOPLAM</td>
      <td>65 TL</td>
    </tr>
  </tfoot>
</table>

İpucu: tbody tr:nth-child(even) ile tek/çift satırlara farklı arka plan rengi verip "zebra tablo" efekti yapabilirsiniz.

Table Borders

<style>
  table, td, th { border: 1px solid black; border-collapse: collapse; }
  th { background-color: #4CAF50; color: white; }
</style>
<table>
  <tr><th>Ürün</th><th>Fiyat</th></tr>
  <tr><td>Elma</td><td>5 TL</td></tr>
</table>

Padding & Spacing

Padding

Hücre içeriği ile hücre kenarlığı arasındaki boşluk.

th, td { padding: 15px; }
Border Spacing

Hücrelerin birbirinden ne kadar ayrı duracağını belirler.

table { border-spacing: 10px; }

Colspan & Rowspan

  • colspan="N": Hücreyi N sütun genişliğine yayar (yatayda birleştirme).
  • rowspan="N": Hücreyi N satır yüksekliğine yayar (dikeyde birleştirme).
<style>table,th,td{border:1px solid black;border-collapse:collapse;padding:8px;}</style>
<table>
  <tr>
    <th colspan="2">Ad Soyad (2 sütunu kapsar)</th>
    <th>Yaş</th>
  </tr>
  <tr>
    <td>Umut</td><td>Yılmaz</td>
    <td rowspan="2">21 (2 satırı kapsar)</td>
  </tr>
  <tr><td>Ayşe</td><td>Kaya</td></tr>
</table>

Colgroup — Sütun Gruplandırma

<colgroup> ve <col> etiketleri, belirli sütunlara toplu CSS uygulamak için kullanılır. <table> açıldıktan hemen sonra yazılır.

<style>table,th,td{border:1px solid black;border-collapse:collapse;padding:8px;}</style>
<table>
  <colgroup>
    <col style="background-color:#fffde7;">
    <col style="background-color:#e3f2fd;">
    <col>
  </colgroup>
  <tr><th>Ad</th><th>Şehir</th><th>Puan</th></tr>
  <tr><td>Umut</td><td>İstanbul</td><td>95</td></tr>
</table>

NOT: <col> ile yalnızca background, width, border ve visibility kullanılabilir.

Lists

Listeler iç içe yerleştirilebilir (nested list). CSS ile yan yana da düzenlenebilir.

Unordered — Sırasız Listeler

list-style-type ile işaret şekli değiştirilebilir: disc (●), circle (○), square (■), none.

<ul style="list-style-type:square;">
  <li>Elma</li>
  <li>Armut</li>
  <li>Muz</li>
</ul>

Ordered — Sıralı Listeler

  • type="A" → A, B, C / type="a" → a, b, c
  • type="I" → Romen rakamları (I, II, III)
  • start="5" → Numaralamayı 5'ten başlatır
<ol type="A" start="3">
  <li>Üçüncü seçenek (C'den başlar)</li>
  <li>Dördüncü seçenek</li>
</ol>

Description — Açıklama Listeleri

Terim–tanım çiftleri için kullanılır. Sözlük ve SSS sayfaları için idealdir.

  • <dl>: Listeyi başlatır.
  • <dt>: Terimi yazar (kalın görünür).
  • <dd>: Terimin açıklamasını yazar, biraz içeriden başlar.
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language — web sayfalarının iskeletini oluşturur.</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets — web sayfalarını görsel olarak şekillendirir.</dd>
</dl>
Block & Inline
Block Level

Her zaman yeni satırdan başlarlar ve satırın tamamını kaplarlar.

Örnekler: div, h1–h6, p, ul, header, footer, section, nav, article, form...

Inline

Yeni satır açmazlar, metnin akışı içinde devam ederler.

Örnekler: span, a, img, strong, b, em, label, button, input...

<div style="background:lightblue; padding:8px;">Ben block</div>
<div style="background:lightgreen; padding:8px;">Ben de block</div>

<p>Bugün hava
  <span style="color:red; font-weight:bold;">çok sıcak</span>
  olacak.
</p>

Semantik Elementler Nedir?

Semantik kelimesi "anlamlı" demektir. <div> ya da <span> gibi etiketler tarayıcıya içerik hakkında hiçbir bilgi vermez — sadece bir kutu ya da sarmalayıcıdır. Ama <header>, <nav>, <article> gibi semantik etiketler hem tarayıcıya hem Google'a hem de ekran okuyuculara "bu bölüm ne işe yarıyor?" bilgisini verir. Bu da SEO'yu ve erişilebilirliği artırır.

Block Elementleri

<div>

En genel block kapsayıcıdır. Semantik anlam taşımaz. Sadece elementleri gruplamak veya CSS/JS uygulamak için kullanılır.

<div style="background:#f0f4ff; padding:16px; border-radius:8px;">
  <p>Bu div içinde bir paragraf.</p>
  <p>Ve bir tane daha.</p>
</div>
<header>

Sayfanın ya da bir bölümün üst kısmını temsil eder. Logo, site adı, ana navigasyon gibi içerikler buraya girer. Sayfada birden fazla olabilir (örneğin her <article>'ın kendi header'ı).

<header style="background:#333; color:white; padding:16px;">
  <h1>Web Tasarım Notları</h1>
  <p>Umut'un ders notları</p>
</header>
<footer>

Sayfanın ya da bir bölümün alt kısmını temsil eder. Telif hakkı, iletişim linkleri, sosyal medya gibi bilgiler buraya girer.

<footer style="background:#222; color:#aaa; padding:16px; text-align:center;">
  <p>© 2025 Umut. Tüm hakları saklıdır.</p>
</footer>
<nav>

Sayfanın navigasyon (menü) bölümünü temsil eder. İçinde genellikle linkler bulunur. Ekran okuyucular bu bölümü "navigasyon" olarak tanımlar ve atlayabilir.

<nav style="background:#eee; padding:12px;">
  <a href="#anasayfa">Anasayfa</a> |
  <a href="#hakkimda">Hakkımda</a> |
  <a href="#iletisim">İletişim</a>
</nav>
<section>

Sayfayı mantıksal bölümlere ayırır. Her section kendi başlığı olan bağımsız bir konu bloğu gibi düşünülebilir. Örneğin "Hizmetler", "Hakkımızda", "İletişim" bölümleri.

<section style="padding:16px; border:1px solid #ddd; margin:8px 0;">
  <h2>Hizmetlerimiz</h2>
  <p>Web tasarım, grafik tasarım ve SEO hizmetleri sunuyoruz.</p>
</section>

<section style="padding:16px; border:1px solid #ddd; margin:8px 0;">
  <h2>Hakkımızda</h2>
  <p>2015'ten beri sektördeyiz.</p>
</section>
<article>

Sayfanın geri kalanından bağımsız olarak anlam ifade eden, kendi başına yayınlanabilir bir içeriği temsil eder. Blog yazıları, haber makaleleri, yorum kartları için kullanılır.

<article style="border:1px solid #ccc; padding:16px; border-radius:8px;">
  <h2>HTML Nedir?</h2>
  <p>Yazar: Umut | 12 Mayıs 2025</p>
  <p>HTML, web sayfalarının temel yapı taşıdır...</p>
</article>
<address>

İletişim bilgilerini temsil eder. Sadece fiziksel adres değil; e-posta, telephone, sosyal medya gibi bilgiler de yazılabilir. Tarayıcılar bunu genellikle italik gösterir.

<address>
  Yazan: <a href="mailto:umut@example.com">Umut Yılmaz</a><br>
  İstanbul, Türkiye<br>
  Tel: 0555 123 45 67
</address>
<figure> ve <figcaption>

<figure> resim, grafik, kod örneği gibi bağımsız içerikleri sarmalar. <figcaption> ise bunun altına ya da üstüne yazılan açıklama etiketidir.

<figure style="text-align:center;">
  <img src="https://via.placeholder.com/200x120" alt="Örnek">
  <figcaption>Şekil 1: Örnek bir görsel açıklaması.</figcaption>
</figure>
<form>

Kullanıcıdan veri toplamak için kullanılan bölümü temsil eder. İçinde input, button, select gibi form elemanları bulunur. action verinin nereye gönderileceğini, method ise nasıl gönderileceğini belirtir.

<form action="/gonder" method="post" style="padding:12px; border:1px solid #ccc;">
  <label>Adınız: <input type="text" name="ad"></label><br><br>
  <button type="submit">Gönder</button>
</form>
<fieldset> ve <legend>

<fieldset> bir form içindeki ilgili alanları görsel olarak kutucuk içine alır. <legend> ise bu kutucuğun üstüne başlık yazar. İkisi her zaman birlikte kullanılır.

<form>
  <fieldset>
    <legend>Kişisel Bilgiler</legend>
    <label>Ad: <input type="text"></label><br><br>
    <label>Soyad: <input type="text"></label>
  </fieldset>

  <fieldset>
    <legend>İletişim</legend>
    <label>E-posta: <input type="email"></label>
  </fieldset>
</form>

Inline Elementleri

<span>

En genel inline kapsayıcıdır. Semantik anlam taşımaz. Metnin bir kısmına CSS veya JS uygulamak için kullanılır.

<p>Bu metinde <span style="color:red; font-weight:bold;">bu kısım</span> vurguludur.</p>
<label>

Form elemanlarının etiketidir. for özniteliği ile bir input'a bağlanır. Label'a tıklandığında ilgili input otomatik odaklanır — kullanıcı deneyimini artırır.

<label for="eposta">E-posta Adresiniz:</label>
<input type="email" id="eposta" placeholder="ornek@mail.com">
<button>

Tıklanabilir buton oluşturur. İçine yazı, ikon veya HTML koyulabilir. type="button" formu göndermez, sadece JS çalıştırır. type="submit" formu gönderir.

<button type="button" onclick="alert('Tıklandı!')" style="padding:8px 16px; background:#4a90d9; color:white; border:none; border-radius:6px; cursor:pointer;">
  Bana Tıkla
</button>
<select> ve <option>

<select> açılır liste oluşturur. Her seçenek <option> ile yazılır. Bu iki etiket her zaman birlikte kullanılır.

<label for="sehir">Şehir seçin:</label>
<select id="sehir" name="sehir">
  <option value="ist">İstanbul</option>
  <option value="ank" selected>Ankara</option>
  <option value="izm">İzmir</option>
</select>
<input>

Kullanıcıdan veri alan en temel form elemanıdır. type özniteliği ile farklı türlere dönüşür: text, email, password, checkbox, radio, number, date, file...

<input type="text" placeholder="Metin girin"><br><br>
<input type="email" placeholder="E-posta"><br><br>
<input type="number" min="0" max="100" placeholder="0-100"><br><br>
<input type="date">
<textarea>

Çok satırlı metin girişi için kullanılır. rows yüksekliği, cols genişliği belirler. Kullanıcı köşesinden sürükleyerek boyutlandırabilir.

<textarea rows="4" cols="40" placeholder="Mesajınızı yazın..."></textarea>
<script>

JavaScript kodu yazmak ya da harici bir JS dosyası bağlamak için kullanılır. Inline element olarak tanımlanır ama genellikle </body> kapanmadan hemen önce ya da <head> içinde yazılır. Sayfanın yüklenmesini bekletmemek için çoğunlukla en alta koyulur.

<!-- Doğrudan kod yazma -->
<script>
  console.log("Merhaba!");
  alert("Sayfa yüklendi.");
</script>

<!-- Harici dosya bağlama -->
<script src="script.js"></script>
Classes

Class, elementlere etiket yapıştırmak gibidir. Aynı class adını birden fazla elemente vererek hepsine tek seferde aynı stili uygulayabilirsiniz.

Düşünün ki bir okulda "9-A sınıfı"ndaki tüm öğrenciler aynı üniformayı giyiyor. "9-A" class adıdır, öğrenciler HTML elementleridir.

CSS'te class seçicisinin önüne nokta (.) koyulur. JavaScript'te getElementsByClassName() ile bu elementlere ulaşılır.

Class Özniteliği Kullanımı

Class niteliği, elementleri gruplayarak CSS ile ortak tasarım özellikleri tanımlamamızı sağlar.

<!DOCTYPE html>
<html>
<head>
<style>
.city {
  background-color: tomato;
  color: white;
  padding: 10px;
}
</style>
</head>
<body>

<h2>Sınıf (Class) Özniteliği</h2>
<p>"city" sınıf adına sahip elementleri şekillendirmek için CSS kullanın:</p>

<h2 class="city">Londra</h2>
<p>Londra, İngiltere'nin başkentidir.</p>

<h2 class="city">Paris</h2>
<p>Paris, Fransa'nın başkentidir.</p>

<h2 class="city">Tokyo</h2>
<p>Tokyo, Japonya'nın başkentidir.</p>

</body>
</html>

Multiple Classes (Çoklu Class Kullanımı)

Bir HTML elementi birden fazla class adına sahip olabilir. Sınıf isimleri aralarında birer boşluk bırakılarak yazılır.

<!DOCTYPE html>
<html>
<head>
<style>
.city {
  background-color: tomato;
  color: white;
  padding: 10px;
} 

.main {
  text-align: center;
}
</style>
</head>
<body>

<h2>Çoklu Sınıf Kullanımı</h2>
<p>Burada, üç h2 elementinin tümü "city" sınıfına aittir. Ek olarak, Londra aynı zamanda metni ortalayan "main" sınıfına da aittir.</p>

<h2 class="city main">Londra</h2>
<h2 class="city">Paris</h2>
<h2 class="city">Tokyo</h2>

</body>
</html>

JavaScript ile Class Kullanımı

JavaScript, getElementsByClassName() fonksiyonu sayesinde aynı class'a sahip tüm elementlere ulaşabilir ve onları toplu hâlde değiştirebilir. Bu örnekte butona tıklandığında "city" class'ına sahip tüm başlıklar gizleniyor.

getElementsByClassName("city") → "city" class'ına sahip tüm elementleri bir dizi gibi döndürür.
for döngüsü → dizideki her bir elementi tek tek gezer.
style.display = "none" → o elementi sayfadan gizler (CSS'teki display:none ile aynı).

<h2>Butona tıklayın ve şehir başlıklarını gizleyin</h2>
<button onclick="sehirleriGizle()" style="padding:8px 16px; margin-bottom:16px; cursor:pointer;">
  Şehirleri Gizle
</button>

<h2 class="city">London</h2>
<p>London, İngiltere'nin başkentidir.</p>

<h2 class="city">Paris</h2>
<p>Paris, Fransa'nın başkentidir.</p>

<h2 class="city">Tokyo</h2>
<p>Tokyo, Japonya'nın başkentidir.</p>

<script>
  function sehirleriGizle() {
    var sehirler = document.getElementsByClassName("city");
    for (var i = 0; i < sehirler.length; i++) {
      sehirler[i].style.display = "none";
    }
  }
</script>
ID

ID niteliği bir HTML belgesinde benzersiz (unique) olmalıdır. Aynı ada sahip birden fazla ID bulunamaz. CSS'te # işareti ile, JS'te getElementById() ile çağrılır.

Bookmarks (Yer İmleri) ile Sayfa İçi Gezinti

Çok uzun sayfalarda, okuyucunun belirli bölümlere doğrudan atlamasını (jump) sağlamak için ID niteliği hedef nokta olarak kullanılır.

<!DOCTYPE html>
<html>
<body>

<p><a href="#C4">Bölüm 4'e Git</a></p>

<h2 id="C4">Bölüm 4</h2>
<p>Bölüm 4'ün hedef içerik alanı burasıdır.</p>

<p>Bu bölüm içeriği açıklar...</p>

</body>
</html>
Button

HTML'de butonlar tıklanabilir arayüz öğeleri oluşturur. type niteliği ile farklı işlevler (normal buton, formu gönderme, formu sıfırlama) kazanırlar.

Temel Buton ve JavaScript Tetikleme

Buton elementleri, onclick gibi olay öznitelikleri (event attributes) kullanarak JavaScript kodlarını pratik bir şekilde çalıştırabilir.

<!DOCTYPE html>
<html>
<body>

<h1>HTML Buton Kullanımı</h1>
<p>Buton, JavaScript kodunu çalıştırmak için <code>onclick</code> özniteliğini kullanır. Butona tıkladığınızda bir uyarı mesajı görüntülenir.</p>

<button onclick="alert('Merhaba!')">Bana Tıkla</button>

</body>
</html>

Buton Türleri (Button Types)

Form etiketleri içerisinde kullanılan buton türlerinin görevleri şu şekildedir:

  • type="submit": Form içerisine girilen tüm verileri action adresinde belirtilen sunucu sayfasına gönderir.
  • type="reset": Form alanlarına yazılmış olan tüm geçici verileri temizler ve ilk varsayılan haline geri döndürür.
<!DOCTYPE html>
<html>
<body>

<h2>HTML Buton Türleri</h2>
<p>Gönder (submit) butonu form verilerini sunucuya iletir, sıfırla (reset) butonu ise form alanlarını tamamen temizler.</p>

<form action="/action_page.php">
  Adınız: <input type="text" name="fname"><br><br>
  
  <button type="submit">Formu Gönder (Submit)</button>
  <button type="reset">Formu Sıfırla (Reset)</button>
</form>

</body>
</html>
Forms

Formlar kullanıcıdan veri toplamak için kullanılır. Tüm form elemanları <form> etiketinin içine yazılır.

Input Türleri

  • type="text": Tek satırlık metin kutusu.
  • type="password": Şifre kutusu — yazılanlar gizlenir.
  • type="radio": Tek seçimli yuvarlak buton.
  • type="checkbox": Onay kutusu — birden fazlası işaretlenebilir.
  • type="submit": Formu göndermek için buton.
  • name: Sunucuya gönderilen veri anahtarıdır.
<form action="/gonder.php">
  <label for="ad">Adınız:</label><br>
  <input type="text" id="ad" name="ad" placeholder="Adınızı girin">
  <br><br>
  <input type="password" name="sifre" placeholder="Şifreniz">
  <br><br>
  <input type="radio" name="cinsiyet" value="erkek"> Erkek
  <input type="radio" name="cinsiyet" value="kadin"> Kadın
  <br><br>
  <input type="checkbox" name="sozlesme"> Sözleşmeyi kabul ediyorum
  <br><br>
  <input type="submit" value="Kayıt Ol">
</form>

Value Özniteliği

Value: Form girdilerine varsayılan (başlangıç) bir değer atamak veya butonların üzerinde yazacak metni belirlemek amacıyla kullanılır.

<form action="/gonder.php">
  Ad: <input type="text" name="isim" value="Umut Özmen"><br><br>
  <input type="submit" value="Veriyi Gönder">
</form>

Readonly Özniteliği

Readonly (Salt Okunur): Uygulandığı form alanını dışarıdan düzenlenemez (değiştirilemez) hale getirir. Kullanıcı metni seçebilir ve kopyalayabilir ancak yeni bir değer yazamaz.

<form action="/gonder.php">
  Kullanıcı ID: <input type="text" name="userid" value="MIS-2026" readonly><br><br>
  <input type="submit" value="Kaydet">
</form>

Multiple Özniteliği (Çoklu Seçim)

Multiple: Kullanıcının <select> listesinden ya da dosya yükleme alanından tek seferde birden fazla seçim yapabilmesine olanak tanır.

<!DOCTYPE html>
<html>
<body>

<h2>Çoklu Seçime İzin Verme (Multiple)</h2>
<p>Kullanıcının birden fazla değer seçebilmesini sağlamak için "multiple" özniteliğini kullanın.</p>

<form action="/action_page.php">
  <label for="cars">Bir araba seçin:</label>
  <select id="cars" name="cars" size="4" multiple>
    <option value="volvo">Volvo</option>
    <option value="saab">Saab</option>
    <option value="fiat">Fiat</option>
    <option value="audi">Audi</option>
  </select><br><br>
  <input type="submit" value="Gönder">
</form>

<p>Birden fazla seçenek belirlemek için Ctrl (Windows) / Command (Mac) tuşuna basılı tutun.</p>

</body>
</html>

Form Elemanları

<select> ve <option> — Açılır Liste (Dropdown): selected özniteliği eklenen seçenek varsayılan gösterilir.

<select name="sehir">
  <option value="ist">İstanbul</option>
  <option value="ank" selected>Ankara (varsayılan)</option>
  <option value="izm">İzmir</option>
</select>

<textarea> — Çok Satırlı Metin Alanı:

<textarea name="mesaj" rows="5" cols="40"
          placeholder="Mesajınızı buraya yazın..."></textarea>

<fieldset>
  <legend>Kişisel Bilgiler</legend>
  <label>Ad: <input type="text"></label><br><br>
  <label>Soyad: <input type="text"></label>
</fieldset>

<button>: İçine hem yazı hem HTML elementi koyabilirsiniz.

<button type="button" onclick="alert('Merhaba!')">Tıkla!</button>

<datalist> — Önerili Giriş Kutusu: Kullanıcı yazmaya başladığında hazır öneriler sunar.

<input list="diller" name="dil" placeholder="Yazın veya seçin">
<datalist id="diller">
  <option value="Python">
  <option value="JavaScript">
  <option value="Java">
  <option value="C++">
</datalist>
CSS Seçiciler

CSS Yorum Satırı

CSS dosyalarında kod bloklarını açıklamak veya geçici olarak devre dışı bırakmak amacıyla kullanılır. /* ile başlar ve */ ile kapatılır.

/* Bu bir CSS yorum satırıdır */
p {
  color: #333; /* Metin rengini belirler */
}

Element Seçici — Belirtilen etiket türündeki tüm elementleri seçer.

<style>
  p { text-align: center; color: darkblue; }
</style>
<p>Bu tüm p etiketlerini etkiler.</p>
ID Seçici (#)

Sayfada yalnızca bir kez bulunan eşsiz bir elementi seçer. CSS'te önüne # konur.

<style>
  #baslik { color: red; }
</style>
<h1 id="baslik">Kırmızı Başlık</h1>
Class Seçici (.)

Aynı class adına sahip tüm elementleri seçer. CSS'te önüne . konur.

<style>
  .vurgu { color: blue; font-weight: bold; }
</style>
<p class="vurgu">Mavi ve kalın.</p>
Evrensel Seçici (*)

Sayfadaki tüm elementleri seçer. En çok margin ve padding sıfırlamak için kullanılır. Aşağıdaki örnekte tüm elementlerin varsayılan boşlukları sıfırlanıyor, sonra başlık ve paragraf ayrı ayrı stilleniyor:

<style>
  * { margin: 0; padding: 0; box-sizing: border-box; }
  body { font-family: sans-serif; background: #f0f4ff; padding: 20px; }
  h2 { color: #e85d04; margin-bottom: 8px; }
  p  { color: #333; line-height: 1.6; }
</style>

<h2>Evrensel seçici (*) ile sıfırlandı</h2>
<p>Artık tarayıcının varsayılan boşlukları yok. Her şey temiz başlıyor.</p>
Gruplandırma (,)

Farklı elementlere aynı stili tek seferde uygulamak için aralarına virgül koyulur. Kod tekrarını önler.

<style>
  h1, h2, p {
    font-family: Georgia, serif;
    color: #2c3e50;
    margin-bottom: 10px;
  }
  h1 { font-size: 28px; border-bottom: 2px solid #e85d04; }
  h2 { font-size: 20px; color: #e85d04; }
</style>

<h1>Ana Başlık</h1>
<h2>Alt Başlık</h2>
<p>Üçü de aynı font ve renk ailesine ait, ama boyutları farklı.</p>

Öncelik Sırası (Specificity)

CSS'te aynı elemente birden fazla kural yazıldığında hangisi kazanır? Kural basit: ne kadar spesifik o kadar güçlü.

Inline CSS → en güçlü (1000 puan)
ID (#) → 100 puan
Class (.) → 10 puan
Element (p, h1) → 1 puan

Aşağıdaki örnekte aynı paragraf için dört farklı kural yazıldı. Hangi rengin kazandığını çalıştırarak gör:

<style>
  /* Element seçici — 1 puan */
  p { color: gray; }

  /* Class seçici — 10 puan */
  .mavi { color: blue; }

  /* ID seçici — 100 puan */
  #kirmizi { color: red; }
</style>

<!-- Inline CSS — 1000 puan (kazanır) -->
<p id="kirmizi" class="mavi" style="color: green;">
  Hangi renk kazandı? (Cevap: Yeşil — çünkü inline en güçlü)
</p>

<!-- ID kazandığı örnek -->
<p id="kirmizi" class="mavi">
  Inline yok. ID mi class mı? (Cevap: Kırmızı — ID daha güçlü)
</p>

<!-- Sadece class -->
<p class="mavi">
  Sadece class var. (Cevap: Mavi)
</p>

<!-- Hiçbiri yok, element kuralı işliyor -->
<p>
  Hiç class/id yok. (Cevap: Gri — element seçici)
</p>
JavaScript Tanıtım

JavaScript Yorum Satırı

JavaScript dilinde açıklama satırları oluşturmak için iki farklı yöntem kullanılır:

  • Tek Satırlı Yorum: Satır başına çift eğik çizgi // eklenir.
  • Çok Satırlı Yorum: Kod bloğu /* ile başlatılır ve */ ile kapatılır.
// Bu tek satırlık bir JavaScript yorumudur

/*
   Bu ise çok satırlı
   bir JavaScript yorum bloğudur.
*/
console.log("Çalışacak kod alanı");

JavaScript ile HTML İçeriğini Değiştirme

JavaScript, HTML dökümanındaki elementlere erişerek onların içeriklerini, stillerini veya özniteliklerini dinamik olarak değiştirebilir.

<!DOCTYPE html>
<html>
<body>

<h2>Metni Değiştirmek İçin JavaScript Kullanımı</h2>
<p>Bu örnek, id="demo" olan bir HTML elementinin içerisine "Merhaba JavaScript!" metnini yazar:</p>

<p id="demo"></p>

<script>
// document.getElementById() metodu belirtilen ID'ye ait elementi yakalar.
// .innerHTML özelliği ise o elementin içerisindeki HTML/metin yapısını günceller.
document.getElementById("demo").innerHTML = "Merhaba JavaScript!";
