HTML Tanıtım

Nested (İç İçe): HTML elementleri birbirinin içine yerleştirilebilir. Örneğin <body> içinde <p>, onun içinde <strong> olabilir.

Empty (Boş) Elementler: Kapanış etiketi olmayan, kendi kendine yeten elementlerdir. <br>, <img>, <hr> gibi.

Temel Etiketler

1

<!DOCTYPE html> → Tarayıcıya "bu dosya HTML5 standardında yazılmış" der. Sayfanın ilk satırı olmak zorundadır ve büyük/küçük harf fark etmez.

2

<html> → Tüm sayfanın en dış kabıdır. Başka her şey bu etiketin içine yazılır.

3

<head> → Sayfa hakkında meta bilgiler tutar: başlık, dil, favicon, CSS bağlantıları gibi. Kullanıcı bu kısmı görmez.

4

<body> → Kullanıcının tarayıcıda gördüğü tüm içerik (yazılar, resimler, linkler, formlar) bu etiketin içine yazılır.

HTML Linkler

HTML'de bağlantılar <a> (anchor = çapa) etiketi ile oluşturulur. href özniteliği (hyper reference) gidilecek adresi belirtir:

<a href="https://www.w3schools.com">Bu bir linkdir</a>
  • <a> etiketi: "anchor" kelimesinin kısaltmasıdır, geminin çapasına benzetilir — sayfayı bir adrese bağlar.
  • href: "hyper reference" kısaltmasıdır, gidilecek URL'yi içerir.

HTML Images

Resim göstermek için <img> etiketi kullanılır. Kapanış etiketi yoktur (boş element). src resmin yolunu, alt ise resim yüklenemediğinde gösterilecek açıklama metnini belirtir:

<img src="https://via.placeholder.com/150" alt="Örnek resim">
  • src (source): Resmin dosya yolu ya da URL adresi.
  • alt: Resim açıklaması — erişilebilirlik ve SEO için önemlidir.

HTML Headings

  • <hr> Etiketi: Sayfaya yatay bir çizgi çeker. Konular veya bölümler arasında görsel bir geçiş/ayraç oluşturmak için kullanılır. Kapanış etiketi yoktur.
<h1>Birinci Bölüm</h1>
<hr>
<h2>İkinci Bölüm</h2>

HTML Elements ve Öznitelikler

Bir HTML etiketinin birden fazla özniteliği (attribute) olabilir. Öznitelik değerleri her zaman tırnak içinde yazılmalıdır. Öznitelikler açılış etiketinin içine eklenir: <etiket öznitelik="değer">

Öznitelikler

style Özniteliği: Bir HTML elementine doğrudan CSS stil kuralları uygulamak için kullanılır. Renk, boyut, yazı tipi gibi görsel özellikler bu yolla verilebilir.

<p style="color:red; font-size:18px;">Bu yazı kırmızı ve büyük.</p>

lang Özniteliği: Sayfanın dilini tarayıcıya ve arama motorlarına bildirir. Genellikle <html> etiketine eklenir. Ekran okuyucular (görme engelliler için) doğru dili kullanmak için bu özniteliğe bakar.

<html lang="tr">

title Özniteliği: Mouse ile elementin üzerine gelindiğinde küçük bir açıklama baloncuğu (tooltip) gösterir. Kullanıcıya ek bilgi vermek için kullanışlıdır.

<p title="Bu bir ipucu metnidir">
  Farenizi buraya getirin.
</p>

HTML Favicon

Favicon, tarayıcı sekmesinde sitenin adının yanında görünen küçük simgedir. <head> içine eklenir:

<link rel="icon" type="image/x-icon" href="resimler/document.png">
  • rel="icon": Tarayıcıya "Bu dosya bir simge" olduğunu söyler.
  • type="image/x-icon": Dosyanın türünü belirtir (klasik .ico formatı için).
  • href: Simge dosyasının yolunu gösterir.

HTML Comments (Yorum Satırları)

Kodun içine not düşmek için kullanılır. Tarayıcı bu satırları görmezden gelir, kullanıcıya gösterilmez. Kod okunabilirliğini artırmak için çok önemlidir.

<!-- Bu bir yorum satırıdır, tarayıcı bunu göstermez -->
<p>Bu yazı görünür.</p>

HTML Styles

Her CSS özelliğinin sonuna noktalı virgül (;) konur. Birden fazla özellik varsa hepsi aynı tırnak içinde, aralarına noktalı virgül koyarak yazılır.

Temel Style Özellikleri

Arka Plan Rengi — background-color

Elementin arkasını belirtilen renge boyar. Tüm sayfanın arka planını boyamak için <body> etiketine eklenir.

<body style="background-color:powderblue;">
  <p>Arka plan açık mavi oldu.</p>
</body>

Metin Rengi — color

Yazının rengini değiştirir. Renk ismi (blue, red…), HEX kodu (#0000ff) veya RGB değeri ile kullanılabilir.

<h1 style="color:blue;">Mavi Başlık</h1>
<p style="color:#e74c3c;">HEX ile kırmızı yazı</p>

Yazı Tipi — font-family

Yazının görüntülendiği fontu (yazı tipini) değiştirir. Birden fazla font yazılabilir; tarayıcı öncelikle birincisini dener, bulamazsa ikinciye geçer.

<p style="font-family:Georgia, serif;">
  Bu yazı Georgia fontuyla.
</p>

Yazı Boyutu — font-size

Yazının boyutunu belirler. px (piksel, sabit boyut) veya % (varsayılan boyutun yüzdesi) ile yazılır. Örneğin 200%, normal boyutun iki katıdır.

<p style="font-size:14px;">Normal boyut (14px)</p>
<p style="font-size:200%;">İki kat büyük (%200)</p>

Metin Hizalama — text-align

Yazının yatay hizasını belirler: sola (left), sağa (right), ortaya (center) veya iki yana yaslanmış (justify).

<h1 style="text-align:center;">Ortalanmış Başlık</h1>
<p style="text-align:right;">Sağa yaslanmış paragraf</p>

CSS Bağlama Yöntemleri

External CSS (Dış Stil Dosyası)

CSS kuralları ayrı bir .css dosyasına yazılır ve HTML sayfasına <head> içinde bir <link> etiketiyle bağlanır. En yaygın kullanılan yöntemdir çünkü tek CSS dosyasıyla yüzlerce sayfanın stilini yönetebilirsiniz.

<head>
  <link rel="stylesheet" href="stil.css">
</head>

Internal CSS (Sayfa İçi Stil)

CSS kuralları <head> bölümünde <style> etiketi içine yazılır. Yalnızca o sayfayı şekillendirmek istediğinizde kullanılır.

<head>
  <style>
    body { background-color: linen; }
    h1 { color: maroon; }
  </style>
</head>
<body>
  <h1>Başlık</h1>
</body>

Inline CSS (Satır İçi Stil)

CSS kuralları doğrudan etiketin içine style="..." şeklinde yazılır. Sadece o tek elementi etkiler. Küçük, tek seferlik düzeltmeler için uygundur.

<h1 style="color:blue; text-align:center;">
  Bu başlık mavi ve ortalı.
</h1>

Öncelik Sırası: Aynı element için birden fazla stil tanımlanmışsa hangisi geçer? Inline CSS her zaman kazanır. External ve Internal arasında ise hangisi HTML'de daha aşağıda yazılıysa o geçerli olur (son yazılan üstündür).

HTML vs CSS Yorum Satırı Farkı

HTML Yorumu 
<!-- Bu HTML yorumudur -->
CSS Yorumu 
/* Bu CSS yorumudur */

Background Image — Arka Plan Resmi

background-repeat (Tekrarlama)

Arka plan resmi elementten küçükse tarayıcı onu varsayılan olarak fayans gibi döşer. Bu davranışı kontrol etmek için background-repeat kullanılır:

  • no-repeat: Resim yalnızca bir kez gösterilir. (En sık tercih)
  • repeat-x: Yalnızca yatayda tekrarlar.
  • repeat-y: Yalnızca dikeyde tekrarlar.
<body style="
  background-image: url('https://www.w3schools.com/css/img_5terre.jpg');
  background-repeat: no-repeat;
  background-size: cover;
">
  <p style="color:white; font-size:24px;">Arka plan resmi</p>
</body>

background-attachment (Sabitleme)

Sayfayı aşağı kaydırdığınızda arka plan resminin de kayıp kaymayacağını belirler.

  • scroll (Varsayılan): Resim sayfayla birlikte kayar.
  • fixed: Resim ekrana çakılı kalır, sayfa kayarken arka plan hareket etmez. Bu "paralaks efekti" olarak bilinir.

Border Style — Kenarlık Stili

<p style="border: 2px solid black;">solid — düz çizgi</p>
<p style="border: 2px dotted black;">dotted — noktalı</p>
<p style="border: 2px dashed black;">dashed — kesik kesik</p>
<p style="border: 4px double black;">double — çift çizgi</p>

Margins & Padding

Margin (Dış Boşluk)

Elementin dışında, komşu elementlerle arasındaki boşluktur. Elementlerin birbirine yapışmasını önler. Tamamen şeffaftır, renklendirilemez.

p { margin: 20px; }
Padding (İç Boşluk)

Elementin içinde, içerik ile kenarlık arasındaki boşluktur. Elementi "nefes alır" hissi verir ve elementin arka plan rengi bu alana da yansır.

p { padding: 15px; }
<p style="margin:30px; padding:20px; background:lightblue; border:2px solid navy;">
  Bu kutunun içi padding, dışı margin.
</p>

HTML Text Formatting

Kalın Yazı Etiketleri

<b> (Bold)

Yazıyı görsel olarak kalın yapar. Anlam taşımaz, salt tasarım amacıyla kullanılır.

<strong>

Yazıyı kalın yapar ve bu metnin önemli olduğunu anlamsal olarak belirtir. Arama motorları ve ekran okuyucular buna önem verir.

<p>Bu <b>kalın ama önemsiz</b> metin.</p>
<p>Bu <strong>kalın ve önemli</strong> metin.</p>

Eğik (İtalik) Yazı Etiketleri

<i> (Italic)

Yazıyı görsel olarak eğik yapar. Genellikle teknik terimler, yabancı kelimeler veya iç sesler için kullanılır.

<em> (Emphasis)

Yazıyı eğik yapar ve o kelimeye vurgu katar. "Bu kelimeyi sesli okurken vurgula" anlamı taşır.

<p><i>İtalik</i> görünüm, <em>vurgulu</em> anlam.</p>

Küçük ve İşaretli Metinler

<small>

Çevreleyen metne oranla yazıyı bir adım daha küçük gösterir. Dipnotlar, telif hakkı bilgileri ve ince yazı için idealdir.

<mark>

Metni fosforlu kalem ile işaretlenmiş gibi sarı arka planla vurgular. Önemli bir terimi ya da arama sonucunu öne çıkarmak için kullanılır.

<p>Normal metin <small>küçük dipnot</small></p>
<p>Sınavda <mark>bu konu çıkacak</mark> dedi hoca.</p>

Silinmiş ve Eklenmiş Metinler

<del> (Deleted)

Metnin üstüne çizgi çeker. Artık geçerli olmayan bilgileri ya da silinmiş içeriği göstermek için kullanılır (örn: indirimli fiyat gösterimi).

<ins> (Inserted)

Metnin altını çizer. Bir belgeye sonradan eklenen, yeni gelen bilgileri işaret etmek için kullanılır.

<p>Fiyat: <del>250 TL</del> <ins>199 TL</ins></p>

Alt ve Üst Simgeler

<sub> (Subscript)

Yazıyı satırın biraz altına ve daha küçük konumlandırır. Kimyasal formüllerde kullanılır. (H₂O'daki 2 gibi)

<sup> (Superscript)

Yazıyı satırın biraz üstüne konumlandırır. Matematik ve üs almada kullanılır. (2³ ya da m² gibi)

<p>Su: H<sub>2</sub>O</p>
<p>Alan: 5 m<sup>2</sup></p>
<p>2<sup>10</sup> = 1024</p>

HTML Quotations (Alıntılar)

<blockquote> — Blok Alıntı

Başka bir kaynaktan alınan uzun bir pasajı veya alıntıyı belirtmek için kullanılır. Tarayıcılar bu elementi genellikle sağdan ve soldan içeriden (indent) gösterir. cite özniteliğine kaynak URL'si yazılabilir.

<blockquote cite="https://tr.wikiquote.org">
  Hayatta en hakiki mürşit ilimdir.
</blockquote>

<q> — Kısa Satır İçi Alıntı

Cümle içinde geçen kısa alıntılar için kullanılır. Tarayıcı, metnin başına ve sonuna otomatik olarak tırnak işareti ekler.

<p>Atatürk şöyle demiştir:
  <q>Hayatta en hakiki mürşit ilimdir.</q>
</p>

<abbr> — Kısaltma

HTML, NATO, WHO gibi kısaltmaların tam halini belirtmek için kullanılır. title özniteliğine tam açıklama yazılır; farenizi kısaltmanın üzerine getirince tooltip olarak görünür. Tarayıcılar genellikle kısaltmanın altına noktalı çizgi çizer.

<p>
  <abbr title="World Health Organization">WHO</abbr>
  1948'de kuruldu.
</p>

<bdo> — Metin Yönünü Değiştirme

"Bi-Directional Override" kısaltmasıdır. Metnin akış yönünü tersine çevirmek için kullanılır. Arapça, İbranice gibi sağdan sola yazılan dillerde veya özel görsel efektlerde işe yarar.

<p>Normal: Merhaba Dünya</p>
<p><bdo dir="rtl">Merhaba Dünya</bdo></p>

dir="rtl" → right to left (sağdan sola). dir="ltr" → left to right (soldan sağa, varsayılan).

HTML Colors

Renk, HTML'de üç farklı yerde uygulanır: arka plan (background-color), metin rengi (color) ve kenarlık rengi (border-color).

Kenarlık Rengi (border): Bir elementin etrafına renkli bir çerçeve ekler.

<h1 style="border: 3px solid Tomato;">Kırmızı kenarlıklı başlık</h1>
<p style="border: 2px dashed DodgerBlue;">Mavi kesik kenarlıklı paragraf</p>

Renk Değerleri: RGB, HEX, HSL

RGB

Kırmızı (Red), Yeşil (Green) ve Mavi (Blue) ışığın karışımıyla renk oluşturur. Her değer 0-255 arasında bir sayıdır. Tüm değerler 0 ise siyah, 255 ise beyaz olur.

Örnek: rgb(255, 99, 71) → domates kırmızısı

HEX

Web'de en yaygın kullanılan formattır. # işaretinin ardından 6 karakterlik bir kod gelir. İlk 2 hane kırmızı, ortaki 2 yeşil, son 2 mavi miktarını gösterir.

Örnek: #ff6347 → domates kırmızısı

HSL

Renk Tonu (Hue: 0-360°), Doygunluk (Saturation: %) ve Parlaklık (Lightness: %) değerlerine dayanır. Renk paletleri tasarlarken sezgisel olduğu için tercih edilir. Parlaklık %0 = siyah, %100 = beyaz, %50 = saf renk.

Örnek: hsl(9, 100%, 64%) → domates kırmızısı

<p style="color: rgb(255,99,71);">RGB ile domates rengi</p>
<p style="color: #ff6347;">HEX ile domates rengi</p>
<p style="color: hsl(9,100%,64%);">HSL ile domates rengi</p>

HTML Links

Target Özelliği

Linke tıklandığında sayfanın nerede açılacağını belirtir.

  • _self (Varsayılan): Aynı sekmede açar, mevcut sayfanın üstüne yazar.
  • _blank: Yeni bir sekme veya pencerede açar. (Dış sitelere verilen linklerde en çok bu kullanılır.)
<a href="https://google.com" target="_blank">
  Google'ı yeni sekmede aç
</a>

URL Türleri

Mutlak URL

Başka bir web sitesine giden tam ve eksiksiz adrestir. Mutlaka https:// ile başlar.

Örnek: href="https://www.google.com"

Göreli URL

Kendi sitenizin içindeki başka bir sayfaya giden kısaltılmış adrestir. https:// yazmanıza gerek yoktur, çünkü kendi sitenizdeyken tarayıcı hangi sunucudan aldığını zaten bilir.

Örnek: href="iletisim.html"

Varsayılan Link Renkleri

CSS ile müdahale edilmezse tarayıcılar linkleri şu standart renklerde gösterir:

  • Ziyaret edilmemiş link: Mavi ve altı çizili.
  • Ziyaret edilmiş link: Mor ve altı çizili.
  • Aktif link (tıklama anında): Kırmızı ve altı çizili.
<a href="https://example.com">Beni tıkla</a>

HTML Bookmarks (Sayfa İçi Bağlantılar)

Uzun sayfalarda belirli bir bölüme doğrudan atlamak için kullanılır. İki adımda yapılır:

1. Adım: Hedef bölüme benzersiz bir id ver:

<h2 id="iletisim">İletişim Bölümü</h2>

2. Adım: O hedefe #id ile link ver:

<a href="#iletisim">İletişime Git</a>

Başka sayfadaki bir bölüme gitmek için:

<!-- Hedef sayfada: -->
<h2 id="galeri">Galeri</h2>

<!-- Bu sayfada: -->
<a href="hakkimizda.html#galeri">Galeriye Git</a>

HTML Images

Image Map — Tıklanabilir Resim Haritası

Tek bir resmin farklı bölgelerine farklı linkler atamak için kullanılır. Örneğin bir Türkiye haritasında her ile ayrı link verebilirsiniz. Üç temel etiket birlikte çalışır:

1

<img usemap="#isim">: Resmi gösterir ve bir haritaya bağlar.

2

<map name="isim">: Haritayı tanımlar. name değeri, usemap'teki isimle aynı olmalı (# olmadan).

3

<area>: Resmin içindeki tıklanabilir bölgeleri, şekillerini ve koordinatlarını tanımlar.

<img src="https://www.w3schools.com/html/workplace.jpg"
     usemap="#calisma-alani" alt="Çalışma Alanı"
     width="400" height="379">

<map name="calisma-alani">
  <area shape="rect" coords="34,44,270,350"
        href="https://www.w3schools.com" alt="Bilgisayar" target="_blank">
  <area shape="circle" coords="337,300,44"
        href="https://www.google.com" alt="Kahve" target="_blank">
</map>

Background Image — Arka Plan Resmi: Sayfanın ya da bir elementin arka planını resimle doldurur.

<body style="background-image: url('https://www.w3schools.com/css/img_5terre.jpg'); background-size: cover;">
  <h1 style="color:white;">Arka plan resim</h1>
</body>

Picture Elementi — Duyarlı Resimler

<picture> Nedir? Farklı ekran boyutları için farklı resimler göstermek amacıyla kullanılır. "Ekran küçükse hafif/kırpılmış resmi göster, büyükse tam çözünürlüklüyü" diyebilirsiniz. Buna Responsive (Duyarlı) Tasarım denir. Mobil kullanıcılar gereksiz büyük resimleri indirmek zorunda kalmaz, sayfa daha hızlı açılır.

<picture>
  <source media="(max-width:600px)"
          srcset="https://www.w3schools.com/css/img_5terre_wide.jpg">
  <img src="https://www.w3schools.com/css/img_5terre.jpg"
       alt="Beş Arazi" style="width:100%">
</picture>

HTML Tables

Tablo yapısı iç içe geçen etiketlerle kurulur: <table> → <tr> (satır) → <td> veya <th> (hücre).

1

<table>: Tablonun tamamını saran ana kapsayıcıdır.

2

<tr> (Table Row): Her bir satırı tanımlar. Tabloda kaç satır varsa o kadar <tr> açılır.

3

<td> (Table Data): Satırın içindeki her bir veri hücresini tanımlar.

4

<th> (Table Header): Başlık hücresidir. <td> ile aynı işlevi görür ama tarayıcılar onu otomatik olarak kalın ve ortalanmış gösterir.

<table border="1">
  <tr>
    <th>Ad</th>
    <th>Soyad</th>
    <th>Yaş</th>
  </tr>
  <tr>
    <td>Umut</td>
    <td>Yılmaz</td>
    <td>21</td>
  </tr>
  <tr>
    <td>Ayşe</td>
    <td>Kaya</td>
    <td>23</td>
  </tr>
</table>

Table Borders (Kenarlıklar)

Varsayılan olarak tablolarda görünür bir kenarlık yoktur. CSS ile eklenir ve <style> bloğu tablonun öncesinde de sonrasında da olabilir.

<style>
  table, td, th {
    border: 1px solid black;
    border-collapse: collapse; /* Çift çizgi yerine tek çizgi yapar */
  }
  th {
    background-color: #4CAF50;
    color: white;
  }
</style>
<table>
  <tr><th>Ürün</th><th>Fiyat</th></tr>
  <tr><td>Elma</td><td>5 TL</td></tr>
</table>

Table Padding & Spacing

Padding

Hücre içeriği ile hücre kenarlığı arasındaki boşluktur. Yazıyı daha okunabilir yapar.

th, td { padding: 15px; }
Border Spacing

Hücrelerin birbirinden ne kadar ayrı duracağını belirler. border-collapse: collapse kullanılıyorsa bu özellik etkisizdir.

table { border-spacing: 10px; }

Colspan & Rowspan — Hücre Birleştirme

Bazen bir başlığın birden fazla sütun ya da satır genişliğinde olması gerekir:

  • colspan="N": Hücreyi N sütun genişliğine yayar (yatayda birleştirme).
  • rowspan="N": Hücreyi N satır yüksekliğine yayar (dikeyde birleştirme).
<style>table,th,td{border:1px solid black;border-collapse:collapse;padding:8px;}</style>
<table>
  <tr>
    <th colspan="2">Ad Soyad (2 sütunu kapsar)</th>
    <th>Yaş</th>
  </tr>
  <tr>
    <td>Umut</td>
    <td>Yılmaz</td>
    <td rowspan="2">21 (2 satırı kapsar)</td>
  </tr>
  <tr>
    <td>Ayşe</td>
    <td>Kaya</td>
  </tr>
</table>

Colgroup — Sütun Gruplandırma

<colgroup> ve <col> etiketleri, tablonun belirli sütunlarına toplu CSS uygulamak için kullanılır. Tek tek her satırın <td>'sine stil vermek yerine sütun bazında bir kez tanımlamanızı sağlar. <table> açıldıktan hemen sonra, <tr> başlamadan önce yazılır.

<style>table,th,td{border:1px solid black;border-collapse:collapse;padding:8px;}</style>
<table>
  <colgroup>
    <col style="background-color:#fffde7;">
    <col style="background-color:#e3f2fd;">
    <col>
  </colgroup>
  <tr><th>Ad</th><th>Şehir</th><th>Puan</th></tr>
  <tr><td>Umut</td><td>İstanbul</td><td>95</td></tr>
  <tr><td>Ayşe</td><td>Ankara</td><td>87</td></tr>
</table>

NOT: <col> ile kullanabileceğiniz CSS özellikleri sınırlıdır: background, width, border ve visibility. Diğer özellikler (color, font gibi) sütun bazında çalışmaz.

HTML Lists

Listeler iç içe yerleştirilebilir (nested list). Listeler yan yana da düzenlenebilir (CSS ile).

Unordered — Sırasız Listeler

Maddelerin sırası önemli olmadığında kullanılır. Her maddenin önüne otomatik bir işaret (bullet) gelir.

  • <ul>: Listeyi başlatır. list-style-type özelliğiyle işaret şekli değiştirilebilir:
    • disc (varsayılan): Dolu daire ●
    • circle: İçi boş daire ○
    • square: Dolu kare ■
    • none: İşaret yok (menü yaparken sıkça kullanılır)
  • <li>: Her bir liste maddesini temsil eder.
<ul style="list-style-type:square;">
  <li>Elma</li>
  <li>Armut</li>
  <li>Muz</li>
</ul>

Ordered — Sıralı Listeler

Maddelerin sırası önemli olduğunda kullanılır. Her maddenin önüne otomatik bir numara veya harf gelir.

  • <ol>: Listeyi başlatır.
    • type="A" → A, B, C şeklinde harf sıralama
    • type="a" → a, b, c şeklinde küçük harf
    • type="I" → Romen rakamları (I, II, III)
    • start="5" → Numaralamayı 5'ten başlatır
  • <li>: Her bir madde.
<ol type="A" start="3">
  <li>Üçüncü seçenek (C'den başlar)</li>
  <li>Dördüncü seçenek</li>
</ol>

Description — Açıklama Listeleri

Terim–tanım çiftleri için kullanılır. Sözlük, glossary veya SSS sayfaları için idealdir.

  • <dl>: Listeyi başlatır (Description List).
  • <dt>: Terimi yazar (Description Term), kalın görünür.
  • <dd>: Terimin açıklamasını yazar, biraz içeriden başlar.
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language — web sayfalarının iskeletini oluşturur.</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets — web sayfalarını görsel olarak şekillendirir.</dd>
</dl>

HTML Block and Inline

Block Seviyeli Elementler

Her zaman yeni satırdan başlarlar ve bulundukları satırın tamamını soldan sağa kaplarlar. "Bencil kutu" gibidir; yanına kimseyi almaz.

  • Üstten ve alttan otomatik boşlukları olabilir.
  • Genişliklerini CSS ile değiştirebilirsiniz.

Örnekler: div, h1…h6, p, ul, li, header, footer, section, article

Satır İçi (Inline) Elementler

Yeni satır açmazlar, metnin akışı içinde devam ederler. Yalnızca kendi içerikleri kadar yer kaplarlar.

  • Genişlik ve yükseklik atanamaz (CSS'te farklı gösterilebilir).

Örnekler: span, a, img, strong, b, em, i, code, mark

<!-- Block örneği: her div yeni satırda -->
<div style="background:lightblue; padding:8px;">Ben block</div>
<div style="background:lightgreen; padding:8px;">Ben de block</div>

<!-- Inline örneği: span'lar yan yana -->
<p>
  Bugün hava
  <span style="color:red; font-weight:bold;">çok sıcak</span>
  olacak.
</p>

HTML Classes

Class, elementlere etiket yapıştırmak gibidir. Aynı class adını birden fazla elemente vererek hepsine tek seferde aynı stili uygulayabilirsiniz.

Düşünün ki bir okulda "9-A sınıfı"ndaki tüm öğrenciler aynı üniformayı giyiyor. Burada "9-A" class adıdır, öğrenciler HTML elementleridir. CSS'te hangi "üniformanın" nasıl görüneceğini siz belirlersiniz.

CSS'te class seçicisinin önüne nokta (.) koyulur. JavaScript'te ise getElementsByClassName() ile bu elementlere ulaşılır.

<style>
  .onemli { color: red; font-weight: bold; }
  .mavi-kutu { background: lightblue; padding: 10px; border-radius: 6px; }
</style>

<p class="onemli">Bu önemli bir mesajdır.</p>
<p class="mavi-kutu">Bu mavi kutudur.</p>
<p class="onemli mavi-kutu">Bu her ikisi de. (Birden fazla class)</p>

HTML Forms

Formlar kullanıcıdan veri toplamak için kullanılır: giriş yapma, kayıt, arama, sipariş gibi. Tüm form elemanları <form> etiketinin içine yazılır.

Input Türleri

<input> formun en temel elemanıdır. type özniteliği ne yazıldığına göre farklı bir giriş alanına dönüşür:

  • type="text": Tek satırlık metin kutusu.
  • type="password": Şifre kutusu — yazılanlar gizlenir.
  • type="radio": Tek seçimli yuvarlak buton — seçenekler arasından yalnızca biri seçilebilir.
  • type="checkbox": Onay kutusu — birden fazlası işaretlenebilir.
  • type="submit": Formu göndermek için buton oluşturur.
  • name: Sunucuya gönderilen veri anahtarıdır. URL'de ?name=değer şeklinde görünür.
<form>
  <label for="ad">Adınız:</label><br>
  <input type="text" id="ad" name="ad" placeholder="Adınızı girin">
  <br><br>

  <label for="sifre">Şifre:</label><br>
  <input type="password" id="sifre" name="sifre">
  <br><br>

  <input type="radio" name="cinsiyet" value="erkek"> Erkek
  <input type="radio" name="cinsiyet" value="kadin"> Kadın
  <br><br>

  <input type="checkbox" name="sozlesme"> Sözleşmeyi kabul ediyorum
  <br><br>

  <input type="submit" value="Kayıt Ol">
</form>

Form Elemanları

<select> ve <option> — Açılır Liste (Dropdown)

Kullanıcının önceden hazırlanmış seçenekler arasından birini seçmesini sağlar. selected özniteliği eklenen seçenek varsayılan olarak gösterilir.

<label>Şehir seçin:</label><br>
<select name="sehir">
  <option value="ist">İstanbul</option>
  <option value="ank" selected>Ankara (varsayılan)</option>
  <option value="izm">İzmir</option>
</select>

<textarea> — Çok Satırlı Metin Alanı

Kullanıcının birden fazla satır metin girebileceği büyük bir alan açar. rows yüksekliği, cols genişliği belirler. Kullanıcı köşesini sürükleyerek boyutu değiştirebilir.

<textarea name="mesaj" rows="5" cols="40"
          placeholder="Mesajınızı buraya yazın..."></textarea>

<fieldset> ve <legend> — Gruplandırma

İlgili form alanlarını görsel olarak bir kutu içine alarak gruplandırır. Uzun formların okunabilirliğini artırır.

<fieldset>
  <legend>Kişisel Bilgiler</legend>
  <label>Ad: <input type="text"></label><br><br>
  <label>Soyad: <input type="text"></label>
</fieldset>

<button> — Buton

Tıklanabilir buton oluşturur. <input type="submit">'den farkı, içine hem yazı hem de HTML elementi (ikon, resim) koyabilmenizdir.

<button type="button" onclick="alert('Merhaba!')">
  Tıkla!
</button>

<datalist> — Önerili Giriş Kutusu

Kullanıcı yazmaya başladığında hazır öneriler sunar; kullanıcı isterse listeden seçer, isterse farklı bir şey yazar. <select>'te yalnızca listeden seçilebilirken, <datalist>'te serbest metin de girilebilir. Bağlantı için <input>'un list değeri ile <datalist>'in id değeri aynı olmalıdır.

<label>Favori Programlama Diliniz:</label><br>
<input list="diller" name="dil" placeholder="Yazın veya seçin">
<datalist id="diller">
  <option value="Python">
  <option value="JavaScript">
  <option value="Java">
  <option value="C++">
</datalist>

CSS — Cascading Style Sheets

CSS Seçiciler

Element Seçici

Belirtilen etiket türündeki tüm elementleri seçer. Örneğin p { } sayfadaki bütün paragrafları etkiler.

<style>
  p { text-align: center; color: darkblue; }
</style>
<p>Bu tüm p etiketlerini etkiler.</p>
<p>Bu da etkilendi.</p>
ID Seçici (#)

Sayfada yalnızca bir kez bulunması gereken eşsiz bir elementi seçer. ID değeri sayfada tekrar edemez. CSS'te önüne # konur.

<style>
  #baslik { color: red; }
</style>
<h1 id="baslik">Kırmızı Başlık</h1>
Class Seçici (.)

Aynı class adına sahip tüm elementleri seçer. Class değeri birden fazla elementte kullanılabilir. CSS'te önüne . konur.

<style>
  .vurgu { color: blue; font-weight: bold; }
</style>
<p>Normal metin.</p>
<p class="vurgu">Mavi ve kalın.</p>
<span class="vurgu">Bu da.</span>
Evrensel Seçici (*)

Sayfadaki tüm elementleri seçer. Genellikle varsayılan margin ve padding sıfırlamak için kullanılır.

<style>
  * { color: purple; font-style: italic; }
</style>
<h2>Başlık</h2>
<p>Paragraf</p>
Gruplandırma Seçicisi (,)

Farklı elementlere aynı stili tek seferde uygulamak için aralarına virgül koyulur. Kod tekrarını azaltır.

<style>
  h1, h2, p { color: green; text-align: center; }
</style>
<h1>Başlık 1</h1>
<h2>Başlık 2</h2>
<p>Paragraf</p>

Öncelik Sırası (Specificity): Aynı element için çakışan kurallar varsa: Inline CSS kazanır. Sonra ID, sonra Class, en son Element seçici gelir. Aynı seviyede iki kural varsa daha sonra yazılan geçerlidir.
